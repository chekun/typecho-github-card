Github Card for Typecho
===================

本插件实现在Typecho中嵌入github的用户或者项目的卡片。

用法很简单：

```
[card]chekun[/card] //潜入用户卡片
```

```
[card]chekun/typecho-github-card[/card] //潜入项目卡片
```

本项目使用了 [lepture/github-cards](https://github.com/lepture/github-cards).
